What is C?

It was created by Dennis Ritchie in 1972.It is the most fundamental language anyone can learn which makes it one of the most popular coding languages.

Why learn C?

It is the most fundamental language anyone can learn today. As most of the languages existing today have some or the other syntax borrowed from C language. So if you learn this it will be beneficial for you to grasp other languages easily. It is faster in execution than Java and Python. It is used to develop UNIX based operating system.

First Program

#include <stdio.h>{
printf("Hello World");
}

The output : Hello World
Syntax

Now let us see some basic syntax :-

  • stdio.h

    It stands for Standard Input Output functions and includes functions like printf, scanf, etc. in the coding.

  • main()

    It is a library function which is the main part of the Program and is executed first.

  • printf()

    It is used to print something on the display.

  • scanf()

    It is used to take input from the user in the run-time.

  • stdlib.h

    It stands for Standard Library functions. It contains information for memory allocation and includes malloc, etc.
